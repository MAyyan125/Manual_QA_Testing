# 🧪 SauceDemo Manual QA Testing Portfolio

## 📌 Project Overview

This repository contains the manual and performance testing artifacts for the **SauceDemo** e-commerce web application.

The purpose of this project is to demonstrate my practical knowledge of Software Quality Assurance (SQA), including test case design, test execution, defect identification, and basic performance testing using Apache JMeter.

---

## 🎯 Objectives

- Perform end-to-end manual functional testing.
- Design and execute comprehensive test cases.
- Validate core application functionality.
- Perform basic non-functional (performance) testing.
- Document testing artifacts following industry best practices.

3. Configure the Thread Group if required.
4. Click **Start** to execute the test.
5. Review the generated reports.
